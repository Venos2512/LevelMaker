# Levels Export Folder

This folder is used to store exported level layouts as JSON files.

## Usage
- Level Exporter tool will save levels here by default
- JSON files can be version controlled
- Share level files with team members
- Keep backups of important levels

## File Naming Convention
Exported levels are automatically named:
```
LevelName_YYYYMMDD_HHmmss.json
```

Example:
```
MyFirstLevel_20260608_143052.json
```
